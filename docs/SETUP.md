# TrustVault — Setup & Deployment

## Prerequisites

- [DFX](https://internetcomputer.org/docs/current/developer-docs/getting-started/install/) `0.30.2+`
- Node.js `>=16`, npm `>=7`
- Python `>=3.10` (benchmarks only)

---

## Local Development

### 1. Install dependencies

```bash
npm install
cd src/frontend && npm install && cd ../..
```

### 2. Start local replica

```bash
dfx start --background
```

> Never use `--clean` after first setup — it wipes canister state.

### 3. Deploy canisters

```bash
dfx deploy
```

This builds the frontend, generates type declarations, and deploys both `credential_backend` and `credential_frontend`.

### 4. Start dev server (hot reload)

```bash
cd src/frontend && npm run dev
```

App runs at `http://localhost:3000`. The deployed canister is available at the URL printed by `dfx deploy`.

### 5. Redeploy after backend changes

```bash
cd src/frontend && npm run build && cd ../.. && dfx deploy
```

---

## Production Build (local canister)

```bash
cd src/frontend && npm run build
cd ../..
dfx deploy --network local
```

Frontend served at: `http://<credential_frontend-canister-id>.localhost:4943/`

---

## Mainnet Deployment (ICP)

You need cycles. Get free cycles at [faucet.dfinity.org](https://faucet.dfinity.org).

```bash
# Check your principal
dfx identity get-principal

# Deploy to mainnet
export DFX_WARNING=-mainnet_plaintext_identity
dfx deploy --network ic
```

App will be live at `https://<credential_frontend-canister-id>.icp0.io`

## Benchmarks

```bash
cd benchmarks
npm install

# Dry run (synthetic data, no canister needed)
node run.js --dry-run --suite all

# Against local canister
node run.js --network local --suite all

# Against mainnet
DFX_NETWORK=ic node run.js --network ic --suite all

# Available suites: scalability | concurrency | finality | stress | all | full
```

### Generate Figures

```bash
cd benchmarks/visualize
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python3 generate_graphs.py
# Figures saved to figures/
```
